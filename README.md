🧠 Aneurysm Detection

This project focuses on detecting intracranial aneurysms from brain imaging data using machine learning and deep learning techniques.

📌 Description

The model processes medical images (such as CTA, MRA, or MRI scans), extracts relevant features, and predicts the presence of aneurysms.
It combines preprocessing, feature extraction, and classification into a single pipeline.

⚙️ Features
Medical image preprocessing
Feature extraction using deep learning
Machine learning-based classification
Evaluation using standard metrics (accuracy, F1-score, ROC-AUC)
